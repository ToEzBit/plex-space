# Plex Space

A macOS desktop app for running AI CLI agents (Claude Code, Codex CLI) in configurable terminal grids. See `CONTEXT.md` for the domain glossary and `docs/adr/` for architectural decisions.

## Agent skills

### Issue tracker

Issues and PRDs live as markdown under `.scratch/<feature>/` (local-markdown tracker; this repo has no remote). See `docs/agents/issue-tracker.md`.

### Triage labels

The five canonical triage roles, recorded as a `Status:` line near the top of each issue/PRD file. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
